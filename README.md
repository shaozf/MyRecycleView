# MyRecycleView
这是一个自定义recycleview 集成了头布局和尾布局 可以新增多个有布局和尾布局。同时深度集成了adapter方法。减少了adapter的开发成本。
这只是一个测试的项目，如有任何疑问可及时沟通本人 或者发邮件给我shao.zf@qq.com
